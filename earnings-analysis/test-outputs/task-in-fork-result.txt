## Key Distinction

There are TWO different "Task" concepts:

1. **Task Management Tools** (TaskCreate, TaskList, TaskGet, TaskUpdate)
   - AVAILABLE in forked skill context
   - Used for tracking work items in a to-do list
   - Can create, read, update task items

2. **Task Subagent Tool** (for spawning Explore/Code/etc. agents)
   - NOT AVAILABLE in forked skill context
   - Cannot delegate to specialized subagents
   - Must use Skill tool or direct tools instead
