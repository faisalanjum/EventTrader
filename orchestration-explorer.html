Claude Code Orchestration — 4 core rules: 1. Only Main Session has Agent tool (sub-agent spawner) 2. Sub-agents CANNOT spawn sub-agents (Task→Task nesting BLOCKED since v2.1.47) 3. Skill→Skill chains = ONLY way to add depth (verified up to L3) 4. Forked skills = ONLY sub-context with thinking enabled (hard-coded off elsewhere) Parallelism: Agent tool = parallel, Skill tool = always sequential Coordination: Shared task list (team agents), file IPC, $ARGUMENTS/return values Key unlock: Team-spawned BG agents get TaskCreate (non-team BG agents don't)
